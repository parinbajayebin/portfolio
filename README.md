# 🌿 Parin's Adventure

An interactive, story-driven portfolio inspired by Studio Ghibli, Monument Valley, and Nintendo aesthetics. Explore my journey from game development to AI engineering through immersive worlds, animated storytelling, and project showcases.

- 🎮 Adventure-inspired portfolio experience

- 🎨 Studio Ghibli-inspired visuals and animations
- 📱 Fully responsive design
- ⚡ Smooth interactions and transitions
